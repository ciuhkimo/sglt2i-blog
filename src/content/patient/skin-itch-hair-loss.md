## 一句話回答

**腎友的皮膚癢、乾燥和掉髮很常見，但不是每個症狀都來自腎臟。** 正確的日常照護可以減輕症狀，但你也要知道什麼時候該停止自己處理、讓醫師接手。

---

## 適用對象

CKD 未透析、血液透析（HD）、腹膜透析（PD）的成人病人和家屬。

## 不適用於

- 急性皮膚感染或蜂窩性組織炎
- 疑似嚴重藥物過敏（全身紅皮、水泡、嘴唇眼睛潰爛）→ **立即急診**
- 腎臟移植後的皮膚問題（免疫抑制劑相關，需另外評估）

---

## 為什麼腎友特別容易皮膚癢？

腎臟病的搔癢（CKD-associated pruritus, CKD-aP）**不是單一原因**，而是多個因素同時作用：

| 機轉 | 白話說 |
|---|---|
| Opioid 受體失衡 | 腦內止癢/致癢的天平歪了 |
| 慢性發炎 | 全身低度發炎讓神經更敏感 |
| 尿毒素累積 | 毒素刺激皮膚和神經 |
| 皮膚乾燥 | 汗腺和皮脂腺功能下降 |
| 鈣磷副甲狀腺失衡 | CKD-MBD 加重搔癢 |
| 缺鐵 | 容易被忽略的因素 |

> **這就是為什麼抗組織胺效果通常不好**——因為搔癢的主因不是組織胺。

### 有多常見？

- HD 病人：約**三到四成**有中度以上搔癢（DOPPS 近年資料約 37%）
- PD 病人：大型 PDOPPS 研究約 **43%** 至少中度受搔癢困擾
- CKD 未透析：約 **24%** 有中度以上搔癢
- 部分病人即使常被癢困擾，也不會主動告訴醫療人員

### 癢不只是不舒服

較嚴重的搔癢常與**睡眠變差、情緒困擾、生活品質下降、較多醫療使用、住院與死亡風險增加**有關。這些多半是觀察性研究結果——意思是「不能忽視」，不代表搔癢本身一定直接造成死亡。

---

## 皮膚乾燥：腎友的「基底問題」

HD 病人皮膚乾燥（xerosis）的盛行率約 **54-91%**。原因是汗腺和皮脂腺功能下降、皮膚屏障受損。

### 保濕 SOP：洗完澡 3 分鐘內

1. **溫水**（不用熱水），洗澡 5-10 分鐘以內
2. 用**無皂、無香精、低敏的清潔劑**，不需全身打肥皂
3. 洗完**3 分鐘內**趁皮膚還濕就擦保濕劑（「soak and seal」）
4. 嚴重乾燥：**軟膏 > 乳霜 > 乳液**

### 有效的成分

凡士林、神經醯胺（ceramide）、甘油、玻尿酸、尿素 5-10%（破皮處勿用）

### 要避免的

- 含香精、酒精的化妝水
- 薄荷、樟腦類產品塗在破皮處
- 來路不明的「萬用藥膏」（常含強效類固醇，長期會皮膚萎縮）
- 中草藥外洗包、精油——「天然」不等於「安全」

## 掉髮：別都怪洗腎

### 最常見的原因

| 原因 | 重點 |
|---|---|
| **休止期落髮** | 住院、感染、開始透析後 2-3 個月才大量掉；誘因解除後 **3-6 個月逐漸改善**，髮量恢復常需 **6-9 個月或更久** |
| **藥物相關** | 某些藥物可能與掉髮相關（heparin、warfarin、ACEi、statin、allopurinol、mycophenolate 等），但**不能自行停藥**——請帶藥單給醫師判斷時間序是否吻合 |
| **缺鐵** | HD 病人 ferritin 在發炎時會被高估，要配合 TSAT 看 |
| **甲狀腺異常** | 低下或亢進都會掉 |
| **雄性禿** | 腎臟病不會「造成」雄性禿，只是同時存在 |
| **蛋白質能量耗損** | 吃太少也會掉髮 |

> **生物素（biotin）補充品在沒有真正缺乏時無效**，而且會干擾 TSH、troponin 等抽血檢驗，讓結果不準確（FDA 曾警告 biotin 可能造成 troponin 假性偏低，延誤心肌梗塞診斷）。若有吃 biotin，抽血前請主動告知醫師與抽血單位。

## 可以跟醫師討論的治療

| 治療 | 證據 / 定位 | 備註 |
|---|---|---|
| **積極保濕** | 基礎第一線 | 對乾燥、皮膚屏障與癢抓循環有幫助；低風險，應普遍執行 |
| **透析充分性優化** | 中 / 情境依賴 | 確認 Kt/V 達標；若懷疑膜材或透析相關反應，可與腎臟科討論調整 |
| **CKD-MBD 控制** | 中 / 間接 | 磷、鈣、PTH 仍應依 CKD-MBD 原則控制；但不能保證止癢 |
| **缺鐵或貧血矯正** | 依貧血指引處理 | 作為掉髮、疲倦與全身症狀評估的一部分；不列為高證據止癢治療 |
| **Gabapentin / Pregabalin** | 高（止癢療效） | CKD-aP 證據較強，但多為 off-label；HD 需低劑量（100-300 mg 僅 HD 後給藥），監測嗜睡、頭暈、跌倒與意識改變 |
| **Difelikefalin（Korsuva）** | 高（限 HD 中重度 CKD-aP） | 成人 HD 病人適用；未建議用於 PD。台灣臨床可近性請以食藥署與院內藥品目錄為準 |
| **窄波 UVB 光照** | 低-中 | 可由皮膚科評估；免疫抑制或皮膚癌高風險者需特別小心 |
| **抗組織胺** | 低 | 對純 CKD-aP 通常效果有限；鎮靜型有時只是幫助睡眠 |

## 什麼時候不能自己處理？紅旗症狀

### 🟡 1-2 週內看腎臟科或皮膚科

- 全身癢沒有皮疹，持續數週未改善
- 癢 + 紅疹、脫屑、結痂
- 掉髮合併圓形禿斑
- 新加藥物後突然癢或起疹

### 🔴 盡快就醫（24-48 小時內）

- **手指縫、手腕、腰、生殖器強烈癢，夜間更糟，家人也癢** → 疑似疥瘡
- **頭皮疤痕、毛孔消失、膿皰、疼痛** → 疤痕性禿髮（毛囊一旦破壞無法再生）
- **癢 + 體重下降、夜間盜汗、淋巴結腫大** → 需排除惡性腫瘤
- **單側沿神經分布的痛 + 水泡** → 帶狀疱疹（72 小時內抗病毒最有效；即使超過 72 小時，仍有新水泡、疼痛明顯或免疫力差者也應盡快就醫）

### 🔴 立即急診

- **全身紅皮脫屑（>90% 體表）**
- **皮膚起水泡 + 嘴唇眼睛潰爛 + 吞嚥痛** → 疑似 SJS/TEN
- **皮疹 + 發燒 + 臉部腫脹 + 用藥 2-8 週後** → 疑似 DRESS 症候群
- **大腿或腹部劇痛 + 紫斑 → 黑色焦痂** → 疑似鈣化防禦（calciphylaxis），高死亡率

## 常見迷思

| 迷思 | 事實 |
|---|---|
| 洗腎皮膚癢都是尿毒造成的 | 透析病人約三到四成有中度以上 CKD 相關搔癢，但不是所有癢都能歸因於腎臟；仍要排除濕疹、疥瘡、藥物疹、肝膽疾病、甲狀腺異常等 |
| 保濕只是化妝品的事 | 保濕是所有指引的**第一線治療**，不是美容 |
| 掉髮代表洗腎沒洗好 | 最常見的是休止期落髮、藥物和缺鐵，跟透析充分性不一定有關 |
| 抗組織胺一定能止癢 | 純尿毒搔癢不是組織胺主導，效果有限 |
| 越用力抓越能止癢 | 抓只會讓癢-抓循環惡化、皮膚增厚、感染 |
| 皮膚乾不過是不好看 | 皮膚乾會降低屏障功能，增加感染和濕疹風險 |
| 吃生物素就能長頭髮 | 非缺乏者無效，且會干擾抽血檢驗結果 |
| 中草藥外洗很天然很安全 | 常見接觸性皮膚炎元凶；腎友皮膚屏障差更敏感 |

## 就醫前請準備

- [ ] 癢的部位、時間（跟透析有沒有關？夜間特別嚴重？）
- [ ] 強度（0-10 分）
- [ ] 是否影響睡眠
- [ ] 手機拍下皮膚或頭皮照片（自然光、含正常皮膚對照）
- [ ] 所有藥物清單（含保健品、外用產品、偏方）
- [ ] 最近 2 個月內新加或停掉的藥
- [ ] 家人或同住者是否有類似症狀
- [ ] 最近的磷、鈣、PTH、Hb、ferritin、TSH
