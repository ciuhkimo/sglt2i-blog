## 一句話回答

**可以，而且值得。** 關鍵不是「能不能去」，而是「有沒有做對規劃」。先確認透析安排，再訂機票。

---

## 適用對象

病況穩定的維持性血液透析（HD）和腹膜透析（PD）成人病人。

## 先和透析團隊討論再規劃

如果近期有住院、通路問題、體液控制不佳、反覆高血鉀、活動性感染或共病不穩定，請先處理好再出發。

---

## 核心原則：先透析，再訂票

| 順序 | 做什麼 |
|---|---|
| 1 | 和腎臟科/透析室討論，確認「可以旅行」 |
| 2 | **預約目的地透析中心**（HD）或安排藥水寄送（PD） |
| 3 | 拿到接收確認後，才付不可退費的機票和住宿 |

---

## HD 旅行規劃

### 預約客座透析

| | 國內 | 國外 |
|---|---|---|
| **提前** | 至少 2-4 週（連假 1 個月） | 至少 6-8 週（熱門地區 3 個月） |
| **管道** | 原透析室護理長協助 + TSN 透析院所查詢 | Fresenius / DaVita / bookdialysis.com / TDQ |
| **健保** | 正常給付，重大傷病免部分負擔 | 事後申請自墊核退（有上限） |

### 要準備的文件（中英文）

- 病摘/透析摘要
- 血清學（1-3 個月內）：HBsAg、anti-HBs、anti-HCV、anti-HIV
- 近期生化（4 週內）：CBC、Na、K、Ca、P、albumin、BUN、Cr
- 心電圖
- 透析處方：模式、血流速、透析液、抗凝劑、乾體重
- 通路資訊：AVF/AVG/CVC、位置、針號
- 用藥清單（學名為主）
- 醫師英文放行信

### 行程排班原則

- **出發前一天做 HD**——把透析間隔留到飛行那天
- 抵達後儘快安排首次透析
- **任何兩次間隔不超過 2-3 天**——漏透析會堆積鉀和水分
- 跨大時差建議到達後先休息 1 天再洗腎

### 通路保護

- 安檢門和 X 光不會傷瘻管
- 通路那隻手：不量血壓、不打針、不提重物、不壓睡
- 每天摸 thrill——失去震顫要立即就醫
- **CVC 病人**：敷料保持乾燥，**禁泡溫泉、泡澡、海水浴**

---

## PD 旅行規劃

### 藥水寄送

- Baxter（Vantive）和 Fresenius 都有全球旅遊計畫
- **國際寄送至少 6-8 週前**安排，部分地區要 3-4 個月
- 藥水寄到飯店或目的地住所
- **機器（cycler）、敷料、口罩、藥物要自己帶**

### APD vs CAPD

| | CAPD | APD |
|---|---|---|
| 設備 | 無機器，較單純 | 需攜帶 cycler |
| 白天彈性 | 需 4 次/日換液 | 夜間機器做，白天較自由 |
| 備案 | — | **多帶 1 組 CAPD 雙連袋**當機器故障備案 |

### 電壓注意

- Baxter HomeChoice **Claria** / **Amia**：100-240V 自動切換，只需轉接頭
- 舊型 HomeChoice：**務必看機背標籤確認電壓**，可能需變壓器
- 台灣 110V/60Hz；日本 100V；歐洲多 220-240V

### 換液環境

密閉房間 + 洗手槽 + 戴口罩 + 洗手 + 不開門窗讓風吹 + 不讓寵物在場

## 藥物與文件（隨身，不要託運）

- [ ] 健保卡、護照
- [ ] 中英文病摘 + 透析處方
- [ ] 用藥清單（學名）+ 過敏清單
- [ ] 足量藥物（旅程 + **至少 7-14 天 buffer**）
- [ ] 疫苗紀錄
- [ ] 緊急聯絡：原透析中心 24h 電話、目的地透析中心、保險服務專線、家屬
- [ ] 「救命卡」：英文標註 ESRD + 透析模式 + 通路側 + 過敏
- [ ] 保險文件 + 健保核退申請書空白

## 飲食水分

- 低鈉低鉀控磷的原則照舊
- **磷結合劑隨餐吃**——包括飛機餐和當地外食
- 機艙乾燥容易渴：用小口、含冰塊、無糖口香糖解渴，**不超過每日水分上限**
- 飲食衛生：瓶裝水（含刷牙）、避免生食、不用路邊冰塊

---

## 健保與保險

### 健保自墊核退

- 健保**不會因出國洗腎而中止**
- 回國後 **6 個月內**申請核退
- 需備：申請書、收據正本、費用明細、診斷書、出入境證明
- 核退金額有上限（依每季公告），**遠低於海外實際費用**

### 旅平險

- 多數旅平險將透析視為**既往症排除**
- 規律透析費用通常**不理賠**
- 旅途中與洗腎無關的新疾病或意外，仍可能涵蓋
- **買保險時明確告知病情，看清除外條款**

---

## 旅途中每天做 3 件事

1. **量體重**——超出乾體重 2-3 kg 要警覺
2. **量血壓**——不在通路那隻手
3. **檢查通路/出口處**——AVF 摸 thrill；PD 看有無紅腫

---

## 旅途緊急狀況

### 🔴 立即就醫

- **PD 透析液混濁 ± 腹痛 ± 發燒**——保留混濁透析液袋帶到急診
- **AVF/AVG 失去 thrill、腫痛、變冷**——24-48 小時內越早越好
- **通路出血壓 20 分鐘止不住**——持續壓迫 + 叫當地急救
- **氣促、無法平躺、咳粉紅泡沫痰**——急性肺水腫
- **胸痛、心悸、暈厥**
- **肌肉無力、麻木、心律不整**——疑高血鉀
- **發燒 ≥38°C 合併寒顫**——疑通路感染

### 漏透析怎麼辦？

**不可用降鉀劑替代透析。** 嚴格限水 <1000 mL/日、低鉀飲食，立即找當地透析中心或急診安排緊急透析。

## 常見迷思

| 迷思 | 事實 |
|---|---|
| 洗腎病人不能旅行 | 穩定腎友規劃得當就能旅行 |
| 出國洗腎健保就停了 | 健保不會中止，事後可核退（有上限） |
| PD 藥水沒辦法寄到國外 | Baxter/Fresenius 都有全球旅遊計畫 |
| 短程出去漏一次洗腎沒差 | **最危險的迷思**——漏透析會堆積鉀和水分，可致命 |
| 機場安檢會傷瘻管 | 不會，但請安檢避免在瘻管上反覆按壓 |
| 買了旅平險就萬事 OK | 標準旅平險幾乎都排除既往症，透析費用不理賠 |
| 洗腎病人不能泡溫泉 | CVC 和 PD 導管者避免；成熟瘻管者可在限制下適量 |
